## 📡 `Architecture Overview`

**Phantom-OS** is an interactive cybersecurity portfolio engineered to look, feel, and function like a live SOC dashboard. It eschews traditional web design paradigms in favor of a **"terminal-first, glassmorphism-second"** cybernetic aesthetic, aiming to provide recruiters and engineering teams with an immersive demonstration of Blue Team culture.

<details>
<summary><b>🛠️ Technology Stack (Click to Expand)</b></summary>
<br>

| Layer | Technology | Details |
| :--- | :--- | :--- |
| **Core Layer** | `Semantic HTML5` | Clean, accessible markup |
| **Style Engine** | `Tailwind CSS` | JIT compilation via CDN, custom tokens |
| **Interactive** | `Vanilla JS (ES6+)` | Canvas particle networks, DOM manipulation |
| **Hosting** | `GitHub Pages` | Edge-deployed static site |

> **Zero Dependency Architecture:** The portal intentionally runs without heavy frameworks (React, Next.js) to guarantee instantaneous load times and eliminate supply chain vulnerabilities. It relies entirely on native browser APIs.
</details>

## 🎨 `UI/UX Design System: "Phantom"`

The visual architecture uses a custom token system injected directly into Tailwind CSS. It is built on three core pillars:

| Pillar | Description |
| :--- | :--- |
| 🧊 **Cybernetic Glassmorphism** | Cards and panels utilize heavy background blur (`backdrop-filter: blur(20px)`) layered over a deep void background (`#05080f`), mimicking holographic HUDs. |
| 🟢 **Neon Threat Accents** | `Phantom Green (#00ff88)`: Active status, primary interactive elements.<br>`Phantom Red (#ff2d55)`: Critical alerts.<br>`Phantom Amber (#ffb800)`: Encrypted data, warnings. |
| ⚡ **Kinetic Feedback** | Immediate visual feedback via CSS text glitching, Canvas API particle networks, radar pulse animations, and a simulated 3-second OS Boot Sequence. |
